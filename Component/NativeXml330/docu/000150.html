TCustomXml.ExternalEncoding
TCustomXml
property ExternalEncoding: TsdStringEncoding;
Description
External encoding is valid after loading, and indicates the encoding detected in the external xml document. Internally, all string values are always encoded in UTF8, so if the external stream is Ansi with codepage or UTF16, a conversion is done. When writing to a file/stream, a BOM is generated for the encoding and a conversion is done from UTF8 to this encoding if necessary.
