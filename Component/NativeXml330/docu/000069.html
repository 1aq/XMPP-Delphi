unit NativeXml

This is a small-footprint implementation to read and write XML documents natively from Delpi code. NativeXmlEx has very fast parsing speeds.

You can use this code to read XML documents from files, streams or strings. The load routine generates events that can be used to display load progress on the fly.

Note: any external encoding (ANSI, UTF16, etc) is converted to an internal encoding that is UTF8. NativeXml uses Utf8String as string type internally, and converts from strings with external encoding in the parsing process. When writing, UTtf8String strings are converted to the external encoding strings, if the encoding was set beforehand, or defaults to UTF8 if no encoding was set.
